🔌 API ENDPOINTS YANG DIPERLUKAN (Backend)
================================================================

Pastikan backend sudah provide endpoint ini:

1. GET /api/respons/pengajuan/:id
   - Response: List<ResponsAdmin> untuk pengajuan tertentu

2. GET /api/respons/pengaduan/:id
   - Response: List<ResponsAdmin> untuk pengaduan tertentu

3. POST /api/respons/pengajuan
   - Body: { id_pengajuan, pesan }
   - Response: { data: ResponsAdmin }

4. POST /api/respons/pengaduan
   - Body: { id_pengajuan, pesan }
   - Response: { data: ResponsAdmin }

5. GET /api/pengaduan
   - (sudah ada, tapi pastikan response include status dan catatan_admin)

USER FLOW:
